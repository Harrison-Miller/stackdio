Stackd.io

A modern Cloud provisioning framework

What is Stackd.io?

Stackd.io is a application framework for launching, provisioning, managing and monitoring an organization's usage of Cloud assets. One of core goals for this project was to ensure that any Cloud provider could be supported by a community of professionals writing adapters as plug-and-play applications for Stackd.io.

Why did we build Stackd.io?

It started, as many of these projects do, as an internal initiative to improve how we manage the Amazon instances that we use during development and testing of our own product. We realized that other people might find this framework a tremendous help in their efforts to enter the world of working on the Cloud, or manage their existing Cloud assets.

How did we build Stackd.io?

Salt

Salt is a provisioning tool that is developed in Python that is an evolutionary step in how IT professionals plan, launch, and manage provisioning of machines and/or images in their network.
